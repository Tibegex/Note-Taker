# Note Taker

## Description

This app will allow a user to take notes and save the information to the webpage.

## Installation

Open webpage https://frozen-cove-23998.herokuapp.com/ and click "Get Started".

## Usage

Click "Get Started", click on "Note Title" and type your tille, and click "Note Text" and type in your text. Click the save button in the top right hand corner.
